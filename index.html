About me

I got into programming because I wanted to be a game developer, but destiny brought me into web development.

My career started in 2008 coding some basic templates for projects created with Ruby On Rails.

I later transitioned into a full-time flash developer. During that time I learned a lot about programming in general. Apart from coding interactive elements for websites, I was also able to create games and even published some of them for mobile.

After Flash died off, I went back to the world of web dev, focusing on mastering Javascript.

I like to be independent, so I dabble in backend technologies as well.

Currently I'm focused on working with React, React Native, VueJS, Node and emerging technologies/libraries.

I like a no-bullshit approach to work, tackling problems head-on.
